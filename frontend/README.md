# Chatter---frontend

Steps to run the React Vite app locally:

1. Run `npm i`
2. Run `npm run dev`
3. Navigate to http://localhost:5173/

# Testing

## To run tests

## To run test

```bash
npm run test
```

## To run the test with Vitest's UI

```bash
npm run test:ui
```

## To run the test coverage

```bash
npm run test:coverage
```
